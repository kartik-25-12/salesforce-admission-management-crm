# 🎓 Student Admission Management System
A CRM built on Salesforce using purely declarative tools (no code).

## 📌 Objects Created
- **Student** — stores student personal details
- **Course** — stores course info like seats, fee, department
- **Application** — links Student to Course with auto-number ID

## ⚙️ Features
- Auto Number on Application (APP-0001, APP-0002...)
- Lookup Relationships between objects
- Validation Rule — blocks applications below 45% marks
- Record-Triggered Flow — auto sets status to Under Review
- Reports & Dashboard — visual admission pipeline

## 🛠️ Tools Used
- Salesforce Developer Org
- Object Manager
- Flow Builder
- Report & Dashboard Builder
